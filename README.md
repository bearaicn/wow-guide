# 魔兽世界灯塔

#### 项目介绍

点亮迷雾中的灯塔，为你指引艾泽拉斯的征途。

魔兽世界灯塔是一个基于 VuePress 构建的知识分享网站，致力于为魔兽世界玩家提供全面、实用的游戏指南和攻略。无论你是新手冒险者还是资深玩家，这里都能为你提供有价值的帮助。

#### 技术架构

- **前端框架**: VuePress 2.0
- **构建工具**: Vite
- **开发语言**: Vue 3 + TypeScript
- **样式预处理器**: Sass
- **包管理器**: pnpm

#### 快速开始

1. **安装依赖**
   ```bash
   pnpm install
   ```

2. **本地开发**
   ```bash
   pnpm docs:dev
   ```

3. **构建生产版本**
   ```bash
   pnpm docs:build
   ```
